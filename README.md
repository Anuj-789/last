# 🌱 2. Land Records Finder

A village-wise land records searching system created to simplify access to frequently used land record pages.

Instead of manually entering details repeatedly, this project provides a simple interface where users can select village and person details and directly navigate to the required land record page.

### 🔗 Live Demo

👉 https://anuj-789.github.io/Land-Records/

### ✨ Features

- Village-wise land record searching
- Quick access system for saved records
- Interactive village dropdown interface
- Person-wise record selection
- Mobile responsive design
- Smooth animations
- Direct navigation to Bihar land record portal pages

### Covered Villages

The system includes land record navigation for villages such as:

- रामनगर
- बहुअरी
- बेलटांडी
- रतनपुरवा
- मुडिला
- सोनखर
- रुपवलिया
- नौरंगिया
- नरैनापुर

### 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
